# MindSpace Database

Main schema file:

```text
database/mindspace_schema.sql
```

Import this file in phpMyAdmin or run it with the MySQL client connected to
XAMPP MySQL. The schema creates the `mind_space` database, all application
tables, indexes, relationships, and the dashboard summary view.

The database is designed for:

- Authentication and refresh tokens
- User profiles and settings
- Spaces
- Notes and checklists
- Articles
- Tasks
- Tags
- Files and attachments
- Public feed interactions
- Notifications and reminders
- Favorites, archive, and trash
- Calendar views
- Statistics
- Offline sync
- Backup and restore
- Future AI features
